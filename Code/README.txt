How to run the program?

Please use the following format, use the executable:

CSE6740FinalProject <filename.tsp> -alg [Iter| FI | MST | SA | TwoOpt] -time <cutoff_in_seconds> -seed <random_seed>

Iter: iterated local search
FI: Furthest insertion
SA: Simulated anealing
TwoOpt: obvious
MST: MST heursitics
